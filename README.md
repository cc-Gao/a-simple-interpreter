## 文件的大致说明
|文件名|作用|
|-----|-----|
|main.c|命令行参数的读取,主程序的执行|
|lexer.c|词法分析器|
|parser.c|语法分析器|
|eval.c|AST树各节点的对应eval函数，该部分思路十分混乱！！！|
|env.c|环境表的相关函数|
|lib.c|各部分实现过程的一些小函数|
|type.h|各结构体，联合体的声明|
|其余头文件|对应部分的函数声明|

## testcode的一些说明

1. @@为代码行注释，@* ... *@为代码块注释
2. 各文件开头简单解释该testcode的测试内容，最后的注释为当时测试时我的输入
3. 程序规定的stdin，stdout即为标准输入，标准输出，指定了3个输入输出函数
`print("...")` `readf()` `readi()`

4. mainfun_test为当时答辩时进行的一些主要功能测试。
